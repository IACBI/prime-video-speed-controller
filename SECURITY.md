# Security Policy

## Scope

Prime Video Speed Controller is a local Windows helper. It launches Microsoft Edge with a dedicated profile and connects only to the local debugging endpoint it starts.

## Supported Versions

Only the latest public version is supported.

## Reporting A Vulnerability

Please open a GitHub issue with a clear description, reproduction steps, and expected impact. Do not include private account data, cookies, tokens, or credentials.

## Security Notes

- The project does not collect telemetry.
- The project does not transmit Prime Video account data.
- The project does not bypass DRM.
- The project does not download video content.
